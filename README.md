# Pokemon app

### Check out project on [pocketmonsters.fly.dev/](https://pocketmonsters.fly.dev/)

### Or clone the project and check it out locally

#### Dependencies

- Node
- Pnpm

#### Follow these steps

`git clone https://github.com/KristianElde/pokemon.git`

`cd pokemon`

`pnpm i`

`pnpm dev`

Check out on localhost:3000
